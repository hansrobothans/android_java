# Example Repository

This is an example repository for demonstrating how to display line numbers in GitHub markdown code blocks.

## Example Code

```html {.line-numbers}
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Example Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script>
      console.log('Hello, world!');
    </script>
  </body>
</html>
